# 📧 Email Spam Classification using K-Nearest Neighbors (KNN)

## 📌 Project Overview

This project focuses on building a **Machine Learning classification model** to detect whether an email is **Spam or Not Spam** using the **K-Nearest Neighbors (KNN)** algorithm.

The model analyzes email features and classifies messages into two categories:
- Spam
- Ham (Not Spam)

---

## 🎯 Objectives

- Perform Exploratory Data Analysis (EDA)  
- Preprocess and clean the email dataset  
- Apply feature scaling (important for KNN)  
- Train a KNN classifier  
- Evaluate model performance using classification metrics
